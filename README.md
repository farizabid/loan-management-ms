# Loan Approval System

Enterprise-style loan approval backend system built using Quarkus.

## Features

- JWT Authentication
- Role-based Access Control
- Loan Application
- Loan Submission
- Financial Calculation
- Approval Workflow

## Tech Stack

- Quarkus
- PostgreSQL
- Panache ORM
- JWT
- Maven

## Roles

- ADMIN
- MAKER
- CHECKER
- APPROVER

## Run Project

```bash
mvn quarkus:dev
